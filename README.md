# PongLeaderboard

## Описание
PongLeaderboard е уеб приложение, базирано на ASP.NET Core 8 и SQL Server.  
Потребителите могат да се логнат, да играят Pong игра и да се състезават за най-висок резултат.  
Има **Leaderboard**, който показва топ 10 резултата, и **admin функции** за изтриване на резултати и потребители.

## За кого е
Приложението е предназначено за хора, които искат да се забавляват с класическата Pong игра и да се състезават с приятели или колеги.

## Проблем, който решава
Позволява на потребителите да имат организирана класация на резултатите и да се състезават, като в същото време администраторът може да управлява резултатите и потребителите.

## Функционалности
- Регистрация и Login на потребители
- Pong игра с Canvas и JavaScript
- Записване на резултати в SQL Server база данни
- Leaderboard с топ 10 резултата
- Admin функции: изтриване на резултати и потребители

## Роли в системата
- **User**: може да играе и вижда Leaderboard
- **Admin**: може да изтрива резултати и потребители
- **Guest**: може да гледа Pong страницата, но не може да записва резултати

## Използвани технологии
- **Език**: C#, JavaScript, HTML, CSS  
- **База данни**: SQL Server  
- **Платформа**: ASP.NET Core 8  
- **Инструменти**: Visual Studio 2022, GitHub  

## Как се стартира приложението
1. Отвори проекта в Visual Studio 2022
2. Увери се, че SQL Server работи и имаш база `PongDB` с таблиците `Users` и `Scores`
3. Отвори `appsettings.json` и проверете connection string към SQL Server
4. Build проекта и стартирай с **IIS Express** или **Kestrel**
5. Отиди на `/Auth/Register` за регистрация или `/Auth/Login` за логин
6. След логин, играй Pong на `/Game/Index` и виж Leaderboard на `/Game/Leaderboard`
